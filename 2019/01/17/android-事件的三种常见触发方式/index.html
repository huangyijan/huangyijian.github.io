Android 事件触发的三种常见方式

1、匿名内部类

其实就是平时所用到的直接函数，类似于下面的这种函数形式

    bt1.setOnClickListener(new View.OnClickListener() {
        @Override
        public void onClick(View view) {
            
        }
    });

上面的这种就属于匿名内部类，也是我们常用的一种调用事件的类型，可以对layout里面的界面元素进行事件的监听。

2、 外部类监听事件

这个先看代码：

public class MainActivity extends AppCompatActivity {

    private Button bt1;
    private Button bt2;


    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);
        bt1=(Button)findViewById(R.id.starService);
        bt2=(Button)findViewById(R.id.stopService);
        bt1.setOnClickListener(this);
        bt1.setOnClickListener(new MyOnclickListener(){
            @Override
            public void onClick(View view) {
                super.onClick(view);
                System.out.println("nihao'");
            }
        });
        bt2.setOnClickListener(new MyOnclickListener(){
            @Override
            public void onClick(View view) {
                super.onClick(view);
//                Toast.makeText(MainActivity.this,"调用子类",1).show();
                System.out.println("buhaoba'");
            }
        });


    }

}


//外部类监听点击事件
//OnClickListener这里是有一个接口库

    class MyOnclickListener implements View.OnClickListener{

    // 相当于统一调用方法
        @Override
        public void onClick(View view) {
            Log.i("tag","父类的方法被调用了");
            view.setAlpha(0.5f);
        }
    }

这里是一个比较简单的学习例子，里面的MyOnclickLinstener就是一个外部类，调用了OnClickLinstener的一个接口。一般写在外面用来包装一个统一的外部类，可以实现对都要实现的方法进行一个统一。

3、实现接口实现监听事件

public class MainActivity extends AppCompatActivity implements View.OnClickListener {

    private Button bt1;
    private Button bt2;


    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);
        bt1=(Button)findViewById(R.id.starService);
        bt2=(Button)findViewById(R.id.stopService);
        bt1.setOnClickListener(this);
    }

    @Override
    public void onClick(View view) {
        Log.i("niahoia", "onClick: nihao");
    }
}

直接在外部类实现就可以调用了
